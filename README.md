# Network Port Scanner

A simple Python-based TCP port scanner used to identify open ports on a target host.

## Features
- Scans ports 1–1024
- Identifies open TCP services
- Uses Python socket programming

## Installation
Clone the repository:

git clone https://github.com/yourusername/Network-Port-Ccanner(Python).git

## Usage

python scanner.py

Enter the target domain or IP when prompted.

## Example

Target: scanme.nmap.org  
Open ports detected: 22, 80, 443

## Educational Purpose
This project is built for cybersecurity learning and ethical security testing.
